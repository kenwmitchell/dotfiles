# Installation

    git clone git://github.com/kenwmitchell/dotfiles.git
    cd dotfiles
    source install.sh

# Usage or other hints
## tmux

|Shortcut key | Effect |
|-------------|--------|
|Ctrl+a, [hjkl]|Move panes VIM style|

## VIM
|Shortcut key | Effect|
|--------------|------|
|,,|New leader key|
|,, + [hjkl]| Change panes|
|jj|Escapes edit mode|
